# Парсер заведений без сайта

Скрипт собирает заведения без сайта из:

- `Google Maps`
- `Яндекс.Карт`
- `2ГИС`

На выходе получаем CSV, где слева название заведения, справа адрес, а также дополнительные поля с источниками, поисковыми запросами и ссылками на карточки.

## Что умеет

- выбор страны и поиск по городу или населенному пункту
- режим широкого поиска по всем заведениям общепита без ручного перебора категорий
- объединение дублей между источниками
- экспорт результата в `CSV` с кодировкой `utf-8-sig` для Excel

## Страны

- `Россия`
- `Казахстан`
- `Беларусь`
- `Кыргызстан`
- `Узбекистан`

## Быстрый запуск

```bash
uv run python main.py
```

Скрипт сам предложит выбрать страну, источник и строку поиска по населенному пункту.
Также он спросит лимит кандидатов на запрос и источник: `0` или пустой ввод означают без лимита.

## Запуск через аргументы

```bash
uv run python main.py --country russia --location moscow --provider all
```

Примеры:

```bash
uv run python main.py --country kazakhstan --location almaty --provider google --headless
uv run python main.py --country belarus --location minsk --provider yandex --max-results 20
uv run python main.py --country russia --location balashikha --provider all --queries кафе ресторан
```

## Проверка окружения

```bash
uv run pytest
```

Если Selenium не может скачать ChromeDriver из интернета, проект сначала попробует найти Chrome и `chromedriver` в системе и в кэше `~/.cache/selenium`. Пути можно задать явно:

```bash
CHROME_BINARY=/path/to/chrome CHROMEDRIVER=/path/to/chromedriver uv run python main.py
```

## Параметры

- `--country` — код страны
- `--location` — город или населенный пункт внутри выбранной страны
- `--provider` — `all`, `google`, `yandex`, `2gis`
- `--queries` — свои поисковые запросы через пробел
- `--max-results` — лимит карточек на каждый поисковый запрос и источник; `0` означает без лимита
- `--headless` — скрытый режим браузера

## Важно

- Если `--max-results` не указан, парсер пытается дойти до конца списка и собирает все карточки, которые успевает отдать источник.
- Для `2ГИС` скрытый режим почти всегда упирается в капчу, поэтому для него приложение автоматически открывает обычное окно браузера.
- Если `2ГИС` всё же покажет капчу, скрипт попросит решить её и продолжит работу.
- Если населенный пункт введен свободным текстом и его нет в локальном справочнике, `Google Maps` и `Яндекс.Карты` продолжат работать, а `2ГИС` будет пропущен.
- Парсинг больших городов может идти долго, особенно при `provider=all`.
